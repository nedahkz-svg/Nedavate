# Image assets — drop your real files here

The site currently uses on-brand placeholders. Replace them by adding these files
to `/public` (the components already point at these paths or have a clearly marked
placeholder you can swap to an `<Image>`):

| File | Used in | Brand-guide photo direction |
|------|---------|------------------------------|
| `portrait.jpg` | Hero | **Authority Portrait** — direct-to-camera, soft side window light, slight forward lean, calm expert expression. 4:5 ratio. |
| `about.jpg` | About section | **Deep Work / In Flow** — at your laptop, teal screen glow, 3/4 angle, genuine focus. |
| `logo.svg` | (optional) replace the inline `Logo` component | Official Nedavate infinity-loop + neural-node mark. |
| `project-1.jpg` … | Portfolio cards | Screenshots of real course/dashboard builds. |
| `og-image.jpg` | Social share preview (1200×630) | Add then reference in `app/layout.tsx` openGraph. |

Colour grade (from the guide): lift shadows toward indigo-purple, warm neutral
midtones, clean teal-cool highlights. Wardrobe: navy/indigo/teal/white solids.

Need an AI-generated placeholder portrait? The guide's ready-to-use Midjourney
prompt is in `Nedavate_Brand_Guide.pdf` (Visual Identity → Photography).
